Angular Opinionated Popout Window

This project is a fork of Shemesh's more customizable Angular-Popout-Window, a superior option if you require more control over your popout window (especially the positioning). This package, in contrast, is designed to be extremely easy to use - providing fewer options but enabling developers to quickly add popout windows with sensible defaults.

Can handle varied aspect ratios and dark themes

The software will attempt to create a window of the same heighth and width as the element it is wrapping. You can set the color of the pop-out/in icon to be white, if you are using a dark theme for example.

Look how aesthetic!

Can collapse when popped out, or stay strong

In cases when space is at a premium, your users may want the extra space on your UI that popping a component out affords. In some cases however shifting elements may prove more confusing than it's worth, we leave it to you to decide.
